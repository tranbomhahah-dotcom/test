## 🚀 Hướng dẫn cài đặt và chạy dự án (Dành cho Giảng viên & Hội đồng)

### 1. Chuẩn bị môi trường
* Cài đặt **JDK 17** (hoặc phiên bản nhóm bạn sử dụng).
* Cài đặt hệ quản trị CSDL **SQL Server**.
* Công cụ lập trình: IntelliJ IDEA / Eclipse / NetBeans.

### 2. Các bước thiết lập
1.  **Clone dự án về máy:**
    ```bash
    git clone [https://github.com/tranbomhahah-dotcom/test.git](https://github.com/tranbomhahah-dotcom/test.git)
    ```
2.  **Khởi tạo Database:** * Mở SQL Server và chạy file script đính kèm trong thư mục `/database` để tạo bảng và nạp dữ liệu mẫu.
    * Cấu hình lại tài khoản, mật khẩu kết nối DB trong file cấu hình dự án của bạn.
3.  **Chạy ứng dụng:**
    * Mở source code bằng IDE, đợi hệ thống tải các thư viện.
    * Tìm đến file chứa hàm `main()` và bấm **Run**.

## 🗂️ Quy định đặt tên commit khi làm việc nhóm
Để lịch sử code của nhóm sạch đẹp, các thành viên thống nhất tuân theo chuẩn:
* `feat: ...` -> Khi thêm một chức năng mới.
* `fix: ...` -> Khi sửa một lỗi/bug trong code.
* `docs: ...` -> Khi cập nhật tài liệu hoặc file README này.
